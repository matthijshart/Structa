# Structa — static site template

Dit is een snelle, statische website-template (HTML/CSS/JS) voor het productconcept “bewakingslaag voor woonverhuur”.
De layout is geïnspireerd door moderne SaaS-landingspagina’s (hero + modules + proof + pricing + FAQ), zonder assets of copy van derden te kopiëren.

## Run lokaal
Open `index.html` in je browser (of gebruik een simpele webserver).

Bijv. met Python:
python -m http.server 8000

Open daarna:
http://localhost:8000/

## Deploy
Deze site werkt op elke static host:
- Netlify
- Vercel (static)
- GitHub Pages

## Aanpassen
- Merknaam / domein: zoek naar “Structa”
- Prijzen: `prijzen.html`
- Modules & copy: `product.html`
- Security & compliance: `security.html`

## Formulier
`contact.html` bevat een demo-formulier als template. Koppel dit aan je backend of een formulierprovider.
